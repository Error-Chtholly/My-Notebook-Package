关于项目

My Notebook Fine V1.0 是一款基于 C# 开发的开源个人记事本软件，旨在为用户提供一个轻量级、可定制的本地文本编辑工具。在日常工作和学习中，许多用户需要一款简单易用的记事本软件来记录信息、整理笔记或编写文本内容，但 Windows 自带的记事本功能较为基础，而专业的代码编辑器又过于复杂。

本项目专注于优化核心文本编辑体验，提供基本的格式管理、文件操作和个性化设置，同时保持软件的简洁性和低资源占用。作为开源项目，它允许用户根据自身需求进行扩展或修改，适用于日常记事、写作或简单的文本处理需求。
